# Overview

As a software engineer, I am focused on mastering the end-to-end development lifecycle, beginning with the establishment of a robust local environment and a seamless version control workflow. My objective is to ensure that my development pipeline is correctly configured for scalability and professional deployment.

This software is a foundational Java application designed to output a standard greeting to the console. It serves as a technical baseline to verify that the compiler, runtime environment, and path configurations are correctly implemented.

The purpose of this project was to validate my technical toolchain, ensuring that code can be written, compiled locally, and successfully pushed to a public repository. This serves as the bedrock for all future, more complex software engineering projects.

[Software Demo Video](https://youtu.be/UHdD3TSRNeM)

# Development Environment

To construct this application, I utilized [Visual Studio Code](https://code.visualstudio.com/download) as the primary Integrated Development Environment (IDE). For version control, I employed [Git](https://git-scm.com/download) locally to manage commits and GitHub for public repository hosting.

The application is written in the Java programming language (OpenJDK 21). Java was selected for its strong typing and consistent execution environment, which ensures that the software can be compiled into bytecode and executed reliably across different platforms.
